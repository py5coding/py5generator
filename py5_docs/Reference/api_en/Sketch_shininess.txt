@@ meta
name = shininess()
category = Lights, Camera
subcategory = Material Properties
type = None
processing_name = shininess

@@ description
Sets the amount of gloss in the surface of shapes. Used in combination with ``ambient()``, ``specular()``, and ``emissive()`` in setting the material properties of shapes.

@@ example
image = shininess_.png

size(100, 100, P3D) 
background(0)
noStroke()
background(0) 
fill(0, 51, 102) 
ambientLight(102, 102, 102)
lightSpecular(204, 204, 204)
directionalLight(102, 102, 102, 0, 0, -1)
specular(255, 255, 255)
translate(30, 50, 0)
shininess(1.0)
sphere(20)  # Left sphere
translate(40, 0, 0) 
shininess(5.0) 
sphere(20)  # Right sphere
