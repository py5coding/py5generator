@@ meta

@@ description
The width of the PShape document.

@@ example
PShape s;

void setup() {
  // The file "bot.svg" must be in the data folder
  // of the current sketch to load successfully
  s = loadShape("bot.svg");
  println(s.width);  // Prints "281.0", the width of the shape
}
